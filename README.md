# data-structures-java-beecrowd

Desenvolvimento de soluções em Java aplicando diferentes estruturas de 
dados a exercícios do [Beecrowd](https://judge.beecrowd.com/pt/profile/1213190).

## Objetivo

Aplicar os conhecimentos teóricos obtidos no estudo das estruturas de 
dados na resolução de problemas reais, praticando tanto a estruturação 
da solução quanto a otimização de código para atender a limites de 
tempo de execução e uso de memória. Faz parte da minha formação em 
Análise e Desenvolvimento de Sistemas, além de servir como prática 
para programação competitiva.

## Organização

Os exercícios estão organizados por tópico. Cada pasta contém o 
código-fonte e um README explicando a lógica da solução.
